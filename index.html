What is Eurorack?

Eurorack is an open modular synthesizer standard where you build your own instrument from independent modules. Instead of one fixed synth architecture, you patch your own: signal generators, tone shapers, timing logic, and control voltages.

Why everyone loves it: instant tactile control, unlimited reconfiguration, and that unique blend of engineering + musical chaos. It feels like wiring a tiny cybernetic lab for sound.

Core Modules: Your Sonic Building Blocks

VCO // Oscillator

The sound source. Creates sine, saw, square, and pulse waveforms.

Think: raw voltage tone engine.

VCF // Filter

Removes or boosts frequencies. Low-pass is the classic subtractive voice.

Think: tone sculptor.

VCA // Amplifier

Controls loudness via CV. Without VCA control, notes often drone forever.

Think: volume gate.

EG // Envelope

Shapes evolution over time (attack/decay/sustain/release).

Think: motion contour.

LFO // Modulator

Slow oscillator for motion: vibrato, wobble, rhythmic sweeps.

Think: automatic performer.

Sequencer

Step-based control voltages and gates for melody and rhythm.

Think: patch brain.

Signal Flow // Animated Current

Classic path: VCO → VCF → VCA → OUTPUT. Modulators (LFO/EG) patch into parameters such as cutoff and amplitude.

Patch Playground // Drag Cables, Hear Routing

Drag from output jacks to input jacks. Routing changes in real time in Web Audio. Start audio, then patch like a mini modular.

Case + Power Basics

  • HP: Horizontal Pitch. Plan width carefully.
  • Current budget: Sum module mA on +12V/-12V/+5V rails.
  • Headroom: Keep ~25% spare for stability and future modules.
  • Depth: Skiffs need shallow modules.

Glossary

CV
Control Voltage; modulation data as voltage.
Gate
On/off control pulse, typically for note duration.
Trigger
Short pulse used to fire an event.
Attenuverter
Scales and optionally inverts incoming modulation.
